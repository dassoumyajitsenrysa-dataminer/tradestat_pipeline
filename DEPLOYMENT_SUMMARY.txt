## âš¡ Quick Deployment (5 minutes)

### 1. Push to GitHub
```bash
cd c:\Users\das.soumyajit\Desktop\tradestat_pipeline
git init
git add .
git commit -m "Initial dashboard"
git remote add origin https://github.com/YOUR-USERNAME/tradestat_pipeline.git
git push -u origin main
```

### 2. Deploy on Streamlit Cloud
- Go to https://share.streamlit.io
- Click "New app"
- Select your GitHub repo
- Set main file to `dashboard/app.py`
- Click "Deploy"

### 3. Add Password Protection (Optional but Recommended)
- Create `.streamlit/secrets.toml` locally (DON'T commit!)
- Add: `[credentials]` and `password = "YourPassword"`
- Update Streamlit Cloud settings with this secret
- Integrate password check in `dashboard/app.py`

### 4. Share with Manager
- Send dashboard URL
- Send password separately via email/secure channel

## ğŸ†˜ Need Help?

### If app won't load:
1. Check Streamlit Cloud logs
2. Verify `requirements.txt` has all dependencies
3. Ensure `dashboard/app.py` exists in root

### If MongoDB connection fails:
1. Use MongoDB Atlas (free tier available)
2. Update connection string in secrets
3. Whitelist Streamlit IP

### If password isn't working:
1. Verify `.streamlit/secrets.toml` exists locally
2. Check password matches in Streamlit Cloud settings
3. Clear browser cache

### For other issues:
